# CMIS242-Project-1

This program takes a file of weights in a format int1, int2 where int1 represents the pounds and int2 the ounces as a percentage of a pound (ie: 2 ounces represented as 1250).

The program opens a JFileChooser window to select the file, reads the file and stores the weights as objects. It is be able to find the smallest, the largest and the average of the weights.

This professor requires the program be a single .java file that includes all classes and the main method.
